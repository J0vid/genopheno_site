About this project

This webpage is part of a multi-lab collaboration to leverage the data that we've collected across many projects. Think of it like interactive supplemental material.

High-dimensional data are uniquely challenging to work with. There are more things to see in some datasets than any one person has time for. People are also interested in a diverse array of different questions.

The goal of this site is to allow you to pursue your individual interests. Learn how similar craniofacial syndromes differ. See what your favorite developmental pathway does to the shape of the face. Play with our syndrome classifier. It's up to you.

If you're curious about the underlying technology this site runs on, visit my github page. Even the code for this site is up there.
